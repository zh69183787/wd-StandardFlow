jQuery weather 插件开发日志
 
jQuery weather v3.0.0( 2013-03-01 )
+ 此版本开放源代码，允许二次修改
+ 此版本改为纯Javascript版本，不再依赖于PHP.
+ 支持全世界所有城市（以前版本仅支持中国城市）
+ 天气数据存放于CND，速度更快
+ 支持mini版本天气图标
+ 修改程序结构、逻辑，占用内存更少
~ 修复若干Bug
 
jQuery weather v2.1.0( 2012-09-19 )
+ 新增定制指定城市的天气预报（支持800+个中国城市）
+ 新增32万条ip段查询
+ 改用数据库查询城市编号，初次访问速度再次加倍（同一用户、同一天只加载一次）
 
jQuery weather v2.0.0( 2012-09-18 )
+ 新增7套天体预报皮肤
+ 新增天气预报图标拖动功能
+ 新增自动下雨、下雪功能
+ 新增天气预报文字展示功能（点击图标出现）
+ 新增图标可以静止、不移动功能
+ 新增DEMO演示功能
+ 丰富天气预报下雨、下雪图标功能
+ 新增父级id参数配置
+ 新增zIndex参数配置
+ 新增天气预报图标是否自动移动参数配置
+ 新增天气预报图标是否自动下雨、下雪参数配置
+ 新增是否加载jQuery.js参数配置
+ 新增皮肤随机选择参数配置
+ 新增浏览器窗体尺寸变化时、天气预报图标运动轨迹功能
+ 压缩了 jquery.weather.js( 目前仅6k )，文件体积更小
+ 服务器设置了js文件过期时间（Expires）、关闭了FileETag，总体加载速度显著提升
+ 新增天气预报数据客户端缓存方案，极大提高加载速度
+ 改变天气预报获取方式：客户端只加载一次天气数据，直到第二天早晨7点才重新请求天气数据
+ 调整内部结构，内部架构更轻便
+ 扩展了接口，更加方便二次开发
~ 修复一处IE拖拽操作可能出现的错误
~ 重新组织代码，分离了脚本处理IE6下PNG24透明问题功能
~ 重新编写天气预报图的DOM清理机制，性能得到较大的优化，内存占有率明显减少
~ 修复IE下内存泄露问题
~ 大幅度提高在IE浏览器下的性能
~ 修复v.0.1存在的一些细微BUG
